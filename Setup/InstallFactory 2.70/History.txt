2.70 (2000.12.7)

  - 기본 설치 폴더를 지정할 때 사용할 수 있는 폴더 예약어에 설치시 타겟 시스템의
    특정 레지스트리 키에서 정보를 읽어 설치폴더를 결정할 수 있도록 하는 "{Reg}"
    폴더 예약어를 추가해 이미 설치한 프로그램이나 어플리케이션의 업그레이드나
    패치 등의 설치화일을 만들 때 편리하도록 함.

  - 설치시 사용자가 임의로 설치 폴더를 바꿀 수 없도록 하는 기능 추가.

  - 콘텍스트 메뉴를 이용한 설치화일 만들기시 현재 작업 중인 프로젝트를 저장하지
    않던 문제 수정.

  - 기타, 몇 가지 내부적인 기능 개선.

 
2.61 (2000.11.1)

  - 기타 화일 관련 문제 수정.

  - 설치화일의 크기가 정상적인 설치화일의 크기 보다 큰 경우에는 오류로 처리하지
    않도록 했음.

  - SetPath.bat Tool 개선.

  - Some minor improvements.


2.60 (2000.10.25)

  - 설치 완료 후 보일 정보 화일에 Local 정보 화일 외에도 "http:"로 시작하는 
    인터넷 URL 주소도 지정할 수 있도록해 설치 완료시 특정 홈 페이지로 연결할 수
    있도록 함.

  - 기타 화일에 Type Library 화일 등록 기능 추가.

  - 기타 화일의 "OCX 등록" 체크 박스를 더욱 넓은 의미의 이름인 "Server 등록"으로
    바꾸고, 설치화일을 만들 때 등록이 가능한 화일인지 검사하는 기능 추가.

  - "{WinDisk}" 폴더 예약어를 "{WinDrive}"로 변경하고, 설치시 루트 경로(예: C:\)
    로 확장되던 것을 드라이브(예: C:)로 확장되도록 함.

  - "{CommonFiles}"와 "{InstallDrive}", "{InstallFileDrive}" 폴더 예약어 추가.

  - INI 화일의 지정에 사용할 수 있는 예약어에 "{WinIni}" 화일 예약어를 추가해
    타겟 시스템의 Windows 디렉토리에 있는 WIN.INI화일을 대상 INI 화일로 지정할
    때 편리하도록 함.

  - "설치시 실행할 사용자 설치프로그램"과 "설치 완료 후 실행할 프로그램",
    "제거시 실행할 사용자 제거프로그램"으로 타겟 시스템에 있는 화일을 지정했을
    때, 설치/제거시 지정한 화일을 타겟 시스템에서 찾을 수 없을 경우, 타겟 
    시스템의 Windows 시스템 디렉토리, Windows 디렉토리, 현재 작업 디렉토리,
    Path로 설정된 경로에서 지정된 화일을 검색하도록 함.

  - 설치화일과 프로젝트 화일의 포맷 개선.

  - 프로젝트 매니저의 레지스트리 관련 오류 수정.

  - InstallFactory 사용설명서에 "Tips" 추가.

  - 기타, 몇가지 내부적인 기능 개선.


2.51 (2000.10.5)

  - 기타 화일 중 "설치할 화일이 신 버전이면 겹쳐 쓰기" 옵션의 화일은 타겟
    시스템에 있는 화일과 설치할 화일의 버전과 수정 날짜가 동일한 경우 설치하던
    것을 설치하지 않도록 함.

  - Windows 2000/NT에서 현재 작업 디렉토리는 삭제할 수 없는 관계로 제거시 가끔 
    폴더나 단축 아이콘 폴더를 삭제하지 못하던 문제 해결.

  - 사용자 임시 디렉토리 삭제 관련 기능 개선.

  - 화일의 Association을 변경하는 설치일 경우, Windows에 Association 변경
    Notify를 하도록 함.


2.50 (2000.9.28)

  - InstallFactory와 설치화일의 아이콘 변경.

  - 위자드창에 "이전" 버튼 추가.

  - 단축 아이콘의 설치 위치 폴더 예약어 중 "{AppData}"를 없애고 대신 
    "{QuickLaunch}" 폴더 예약어를 추가해 "{AppData}" 폴더 예약어를 사용할 때
    보다 더욱 손쉽게 프로그램을 작업 표시줄의 "빠른 실행"에 등록할 수 있도록
    했고, 단축 아이콘의 명령행 인수와 작업 디렉토리의 지정에 사용할 수 있는
    폴더 예약어에 "{TargetDir}" 폴더 예약어를 추가해 단축 아이콘의 명령행 인수에
    대상이 있는 디렉토리를 삽입할 때와 단축 아이콘의 작업 디렉토리를 대상이 있는
    디렉토리로 지정할 때 편리하도록 함.

  - 단축 아이콘, 기타화일, 레지스트리, INI화일 항목 추가 대화상자에서
    "Ctrl+Enter" 키로 항목을 추가할 수 있도록 함.

  - 이전 버전에서, 이미 설정된 값이 있어서 설치하지 않은 레지스트리와 INI항목은
    제거시 삭제하지 않던 것을 "제거시 이 (키 또는) 값은 제거하지 않음"
    체크박스로 삭제 여부를 지정할 수 있도록 함.

  - InstallFactory 버전 2.30과 2.40이 기타 화일 설치시 설치할 화일과 이미 있는 
    화일의 버전을 제대로 비교하지 못하던 문제와, Windows 95에서 사용자
    설치프로그램 종료시 가끔 연산오류가 발생하던 문제점 해결.

  - Windows 2000/NT 에서 정보 위자드창과 레지스트리 관련 기능 개선.

  - Windows 98/ME/2000 에서 공용 화일 대화상자의 작업 디렉토리 관련 기능 개선.

  - 시스템 재시작 설치 관련 기능 개선.

  - 제거시 삭제 화일 리스트 관련 프로그래밍 실수 수정.

  - 기타 화일을 타겟 시스템에 등록/해제할 때 오류가 발생하면 나오던 시스템 오류
    메세지를 자체 오류 메세지로 바꾸는 등, 설치/제거 중 발생하는 오류에 관한
    레포팅 기능 개선.

  - 기타, 여러 내부적인 기능 개선 및, 몇가지 오류 수정.

2.20 (2000.8.15)

  - InstallFactory 기본 위자드 비트맵 Changed.

  - 배경창의 채움색을 두가지 색으로 지정할 수 있도록 함.

  - 단축 아이콘의 작업 디렉토리를 지정하지 않으면 대상이 있는 폴더로 지정되던

  - 레지스트리 값의 이름과 값을 지정하지 않고 값의 형을 문자열 형으로 했을 때

  - 레지스트리의 "이진 값"의 지정은 16진(Hexa Decimal) 수로만 지정할 수 있도록 
    했고, 16진 숫자앞에 "0x"나 "0X"는 붙이지 않아도 되도록 했음.
     
  - 라이센스와 정보 위자드 창에서, 포커스가 에디트 창에 있지 않아도 Up, Down, 
    PgUp, PgDn 키를 눌러 에디트 창의 내용을 스크롤할 수 있도록 함.

  - 프로젝트 화일의 포맷 수정을 비롯한 여러가지 내부적인 기능 개선.

  - 프로젝트 매니저 추가.
